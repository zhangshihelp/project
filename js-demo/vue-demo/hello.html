安装Vue和使用Vue

安装node.js，然后在cmd中输入node -v和npm -v查看是否安装成功

然后如果有代理设置代理配置，npm config set proxy http://username:password@server:port/

接着输入npm install nrm -g，其中-g代表全局设置

最后nrm ls查看所有的源，nrm use taobao选择淘宝源

在当前项目下的终端中先输入npm init -y，然后输入npm install vue --save
